# ANDROWINOS

ANDROWINOS is a next-generation hybrid operating system that combines
native Android support with Windows application compatibility.

## Vision
One system that brings mobile and desktop worlds together:
- Android apps run natively
- Windows apps and games run through a deeply integrated compatibility layer
- High performance focus for ARM64 devices
- Open-source and community-driven

## Key Goals
- Native Android framework (AOSP-based)
- Integrated Windows compatibility layer (Winlator/Wine-based)
- Desktop-like UI mode
- Optimized gaming performance
- Modular system architecture

## Current Status
 Pre-Alpha — early research & architecture phase

## Roadmap (High Level)
- Environment setup & AOSP builds
- Compatibility layer research
- First bootable ANDROWINOS image
- Developer preview (Beta)
